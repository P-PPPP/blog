RDP 在无显示器的环境下进行gui调试

December 9, 2021 · 1 min · PP | edit

需求#

在WIndows Server 服务器上进行剪映的自动化服务,需要用到 pyautogui 和 win32api

解决#

  • 保持gui链接: 在远程桌面中创建 Close.bat 并输入
   %1 mshta vbscript:CreateObject("Shell.Application").ShellExecute("cmd.exe","/c %~s0 ::","","runas",1)(window.close)&&exit
   for /f "skip=1 tokens=3" %%s in ('query user %USERNAME%') do (
   %windir%\System32\tscon.exe %%s /dest:console
   )

即可在断开RDP的情况下保持Gui 但是会自动最小化务必在代码中把需要调试的gui应用置于前端,否则无法识别
